# 대인관계능력 

## 소개 

**1. 대인관계능력의 의미와 중요성** 

대인관계능력의 의미: 직장생활에서 협조적인 관계를 유지하고, 조직구성원들에게 도움을 줄 수 있으며, 조직 내부 및 외부의 갈등을 원만히 해결하고 고객의 요구를 충족시킬 수 있는 능력 

대인관계 형성 시 가장 중요한 요소: 말이나 행동이 비롯되는 사람됨. 즉 내면 또는 내적 성품 

건강한 대인관계에서 소중한 것: 자신을 존중하고 배려할 수 있을 때 비로소 타인을 존중하고 배려할 수 있음

**2. 대인관계 향상 방법** 

대인관계 향상의 의미: 인간관계에서 구축하는 신뢰의 정도를 높이는 것 

대인관계 향상을 위한 행동(감정은행계좌 예금 적립 수단) 

1) 상대방에 대한 이해와 양보 
2) 사소한 일에 대한 관심
3) 약속의 이행
4) 칭찬하고 감사하는 마음 
5) 언행일치 
6) 진지한 사과(진정성 있는 태도) 

**3. 다양한 대인관계 양식** 

대인관계 양식의 구분 

1) 타인을 마음대로 통제하려는 지배성 차원과 타인을 호의적으로 대하는 정도인 친화성 차원에 따라 총 8개의 유형으로 구분됨 
2) 지배성 차원: 지배-복종의 연속선상에서 대인행동 평가 
3) 친화성 차원: 사랑-미움의 연속선상에서 대인행동 평가 

대인관계 양식의 유형 

| 구분 | 특징 | 보완점 |
| --- | --------------------------------- | ----------------------- |
| 지배형 | - 타인을 통제하려는 경향이 있음 <br > - 지도력과 추진력이 있으나 독단적이고 논쟁적 | - 타인의 의견을 경청하고, 자신의 지배적 욕구를 반성해야 함 | 
| 실리형 | - 실리적 이익을 추구하는 성향 <br> - 자기중심적이고 탕니에 대한 배려가 부족한 | - 타인의 이익을 배려하는 노력이 필요하며, 타인의 신뢰를 얻는 일에 관심을 가져야함 | 
| 냉담형 | - 타인의 감정에 무관심하고 타인과 거리를 두는 경향이 있음 <br> - 이성적이고 의지력이 강하나 따뜻한 감정을 표현하지 못함 | - 타인의 감정 상태에 깊은 관심을 가지고, 긍정적인 감정을 부드럽게 표현하는 연습이 필요함 | 
| 고립형 | - 혼자 있거나 혼자 말하는 것을 좋아하는 유형 <br> - 타인을 두려워하고 사회적으로 고립될 가능성이 있음 | - 대인관계의 중요성을 인식하기 위해 적극적으로 노력해야 하며, 대인관계에서 느끼는 불편함에 대해 깊이 생각해 봐야 함 |
| 복종형 | - 대인관계에서 수동적이고 의존적인 유형 <br> - 자신감이 없고, 타인에게 자신이 원하는 바를 전달하지 못함 | - 대인관계에서 독립성을 키우고, 자기주장을 할 필요가 있음 | 
| 순박형 | - 겸손하고 너그러움 <br> - 원치 않아도 반대하지 못하고, 주관 없이 끌려다닐 가능성이 높음 | - 타인의 의도를 신중히 파악하고, 자신의 의견을 주장하려는 노력이 필요함 | 
| 친화형 | - 자기희생적인 태도로 타인을 배려하는 유형 <br> - 타인의 요구를 잘 거절하지 못하고, 타인을 도와주려고 과도하고 나서는 경향이 있음 | - 타인과의 정서적 거리를 유지하려는 노력이 필요하며, 자신의 이익도 타인의 이익만큼 중요함을 인식해야 함 | 
| 사교형 | - 외향적이고, 타인에게 간섭하는 경향 <br> - 흥분을 잘하고 충동적이며, 혼자서 시간을 보내는 것을 어려워함 | - 혼자만의 내면적 생활에 좀 더 깊은 관심이 필요하며, 타인으로부터 인정받으려는 욕구에 대한 성찰이 필요함 | 

## 팀워크능력 

**1. 팀워크의 의미와 특징** 

팀워크 = Team + Work 

팀워크와 응집력

- 팀워크: 팀 구성원이 공통의 목적을 달성하기 위해 상호 관계성을 가지고 협력하여 업무를 수행하는 것 
- 응집력: 사람들고 하여금 집단에 머물고, 그 집단의 멤버로서 계속 남아 있기를 원하게 만드는 힘 

효과적인 팀워크 

- 훌륭한 팀워크를 위한 팀원의 기본요소 

    - 팀원 간 공동의 목표의식과 강한 도전의식 
    - 팀원 간 상호 신뢰 및 존중 
    - 서로 협력하면서 각자의 역할과 책임 수행 
    - 솔직한 대화로 서로 이해 
    - 강한 자신감으로 상대방의 사기 진작 

- 팀워크를 저해하는 요소 

    - 조직에 대한 이해 부족 
    - 자기중심정인 이기주의 
    - '내가'라는 자아의식의 과잉 
    - 질투나 시기로 인한 파벌주의 
    - 그릇된 우정과 인정 
    - 사고방식의 차이에 대한 무시 

효과적인 팀의 특성 

1) 팀의 사명과 목표를 명확하게 기술함 
2) 창조적으로 운영함 
3) 결과에 초점을 맞춤 
4) 역할과 책임을 명료화함 
5) 조직화가 잘 되어 있음 
6) 개인의 강점을 활용함 
7) 리더십 역량을 공유하며 구성원 상호 간에 지원을 아끼지 않음 
8) 팀 풍토를 발전시킴 
9) 의견 불일치를 건설적으로 해결하 
10) 개방적으로 의사소통함 
11) 객관적인 결정을 내림 
12) 팀 자체의 효과성을 평가함 

팀의 발전 과정 

1) 1단계 형성기: 리더에게 의지, 심각한 논의 회피 
2) 2단계 격동기: 경쟁과 마찰, 리더십 구조-권한-권위에 대한 문제 전반에 걸쳐서 경쟁성과 적대감 
3) 3단계 규범기: 팀원 간 신뢰 형성, 공동체 형성과 팀의 문제해결에 더욱 집주으 단결력 심화, 솔직한 의사소통과 응집력, 창의력과 생산성 왕성 
4) 4단계 성취기: 더 높은 성과 창출 목표, 개방적 소통 통해 상호 신뢰, 지시 없어도 각자 자율적으로 업무 수행
